Summary

The final game used narrative to educate the user on the various museum objects. Blob, an alien, has crash landed on Earth with a broken ship and must rebuild their ship using the various museum artifacts.

Reflection

Humans are storytellers, it is what makes us human. When we want to teach someone about an artefact in a museum, we should not bombard them with facts but rather build a narrative and contextualise the object. Blob's Adventure builds a fictional narrative around uncconected objects in different museums. Each time Blob utilises a different tool the user would learn it's function and recontextualise it's purpose.
